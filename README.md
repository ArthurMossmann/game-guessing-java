🎮 Guessing Game in Java

Project developed in Java focusing on programming logic practice.

🚀 Features

Difficulty menu:

* Easy (0 to 10)

* Medium (0 to 50)

* Hard (0 to 100)

* Input validation (prevents letter errors)
* Hints during the game (higher or lower)
* Attempt counter

🧠 Technologies and concepts

* Java
* Scanner (data input)
* Random (random numbers)
* Repetition structures (`while`)
* Conditional structures (`if`, `switch`)

▶️ How to run

1. Compile:

``` javac src/Adivinhacaogame.java
```

2. Run:

``` java -cp src Adivinhacaogame
```

📌 Objective

To practice programming logic and reinforce basic Java concepts. ---
